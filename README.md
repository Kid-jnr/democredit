# Description 

Lendsqr Backend Engineer Assessment.
Demo Credit is a mobile lending app that requires wallet functionality. This is needed as borrowers need a wallet to receive the loans they have been granted and also send the money for repayments.

# Installation
First install Nest.js globally on your machine
```bash
npm i -g @nestjs/cli
```
Then pull the project from GitHub and run the following
```bash
npm install #to install dependecies 
npm run start:dev #to start the project 
```
Create a .env file and add this line
```bash 
JWT_SECRET = 'secret key'
```
Edit the knexfile.js to match your database settings

Then finally run 
```bash 
npx knex migrate:latest #to migrate your database
npm run test:watch #to run unit test
```
	> Node version is 18.12.1.

# Details
For the database I created 2 tables which are 
- Users 
- Transactions

And I used JWT for authentication. the project consist of 5 endpoints
- Register
- Login
- Deposit
- Withdrawal
- Transfer

Only authenticated users are allowed to access the last 3 endpoints below is a link to the postman documentation 
 [Postman Doc](https://documenter.getpostman.com/view/11075812/2s8ZDR7RVJ)
